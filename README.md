# 알바인 (Albain) – 지역 기반 아르바이트 매칭 플랫폼

React + Spring Boot + MySQL 기반으로 개발한 **구인·구직 아르바이트 매칭 웹 서비스**입니다.  
사용자는 공고를 등록하거나 이력서를 제출할 수 있으며, 지역·직종 등 조건별 검색이 가능합니다.


## 📌 주요 기능

- 이력서 등록 및 수정
- 구인공고 등록 / 수정 / 삭제
- 조건별 검색 기능 (지역, 직종 등)
- 카카오 지도 API 연동 
- 게시글 상세 보기 및 댓글 기능
